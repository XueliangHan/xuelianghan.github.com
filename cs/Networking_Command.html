ARP (Address Resolution Protocol)

  • Design Function - Displays and modifies the IP-to-Physical address translation tables used by address resolution protocol (ARP).
  • Usage of Windows
    • ARP -s inet_addr eth_addr [if_addr]
    • ARP -d inet_addr [if_addr]
    • ARP -a [inet_addr] [-N if_addr] [-v]
  • Parameters of Windows - type arp /?
  • Example C:\Users\HK>arp -a 192.168.1.1 Interface: 192.168.1.101 --- 0xd Internet Address Physical Address Type 192.168.1.1 38-83-45-51-96-2c dynamic
