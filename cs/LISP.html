Absolute Value

  • a.Function of eval absolute value in Scheme way 1

    (define (abs x)
    (cond ((< x 0) (- x))
    ((= x 0) 0)
    ((> x 0) x)))

  • b.Function of eval absolute value in Scheme way 2 (define (abs x)
    (if (< x 0)
    (- x)
    x))
